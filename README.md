## Ian Perez

Software engineer building at the intersection of AI agents, fintech, and Web3. I create tools that give AI agents access to the real world — banking, trading, tax optimization, video editing, prediction markets, and on-chain protocols.

Most of my recent work is in the **Agent Skills Protocol** ecosystem: production-ready skill packages that teach AI coding agents new capabilities. Built for OpenClaw, Claude Code, and Codex.
